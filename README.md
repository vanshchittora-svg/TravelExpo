# TravelExpo
Travel Expo is a full-stack web application developed using PHP and MySQL that allows users to explore popular travel destinations, view detailed information, search for locations, and discover recommended hotels.  
Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- PHP

Database:
- MySQL

Server:
- XAMPP (Apache & MySQL)

## How to Run

1. Install XAMPP
2. Start Apache and MySQL
3. Copy project folder into:
   C:\xampp\htdocs\travel-expo

4. Open phpMyAdmin
5. Create a database (example: travel_expo)
6. Import the database.sql file

7. Open browser and go to:
   http://localhost/travel-expo
